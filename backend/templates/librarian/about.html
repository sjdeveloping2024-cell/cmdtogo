📚 About Pick-A-Book

Pick-A-Book is a full-stack library management system built with Flask, MySQL, Socket.IO, and Expo React Native.

Version: 2.0.0 — Socket.IO Edition

Features: Real-time dashboard updates via WebSocket, book inventory management, student management, borrow/return tracking with due dates, Arduino LCD integration, and a student mobile app.
